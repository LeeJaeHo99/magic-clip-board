Magic Clipboard 개인정보 처리방침

1. 수집하는 정보

Magic Clipboard는 다음과 같은 정보를 수집합니다:

  • 복사된 텍스트: 사용자가 웹페이지에서 복사한 텍스트 내용
  • 복사 시간: 텍스트가 복사된 시간 정보

2. 정보 저장 및 사용

  • 모든 데이터는 사용자의 로컬 기기에만 저장됩니다
  • 외부 서버로 데이터를 전송하지 않습니다
  • 수집된 정보는 클립보드 히스토리 기능 제공 목적으로만 사용됩니다

3. 데이터 보안

  • Chrome의 안전한 로컬 스토리지를 사용합니다
  • 데이터는 사용자 기기에만 저장되며 외부로 전송되지 않습니다
  • 확장 프로그램 제거 시 모든 데이터가 자동으로 삭제됩니다

4. 권한 사용

  • Storage 권한: 복사된 텍스트를 로컬에 저장하기 위해 사용
  • Host 권한: 모든 웹사이트에서 복사 이벤트를 감지하기 위해 사용
